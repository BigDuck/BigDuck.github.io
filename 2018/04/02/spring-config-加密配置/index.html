spring config 加密配置

使用对称加密和非对称加密

非对称加密

1.生成keystore

keytool -genkeypair -alias cloud-config-server -keyalg RSA -keystore cloud-config-server.keystore

根据提示输入参数，最后生成cloud-config-server.keystore 放到config-server的resources目录下。

2.替换jre/lib/security下面的jar包

下载得到local_policy.jar和US_export_policy.jar
替换掉jre/lib/security原有的jar


启动项目测试的链接：
http://localhost:8600/decrypt 解密 http://localhost:8600/encrypt 加密
其中加密密文在配置文件中记得以{cipher}开头，切记加上单引号

使用对称密钥加密

encrypt:
key:'***这里写密钥***'

总结

1.配置非对称加密密文的时候记得以{cipher}开头而且要有单引号
